NODU Mark

"No One Doesn't Understand" - An intuitive markup language that can be understood without a renderer.

NODU Mark Syntax Guide

  • [|Bold Text|]: Bold
  • ~Wavy Text~: Wavy
  • -Strikethrough-: Strikethrough
  • */Italic Text/*: Italic
  • %#hexcode:text%: Color Change
  • #Header#: H1 Header
  • -#Header#-: H2 Header (up to level 20)
  • !!!Slightly Larger Text!!!: Slightly Larger Text
  • ----: Horizontal Line
  • ([Speech Bubble]): Speech Bubble
  • {[Thought Bubble]}: Thought Bubble
  • [!Image Link!]: Display Image
  • - Unordered List: Unordered List
  • !- Ordered List: Ordered List
